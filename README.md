# 🌍 Carbon Emission Tracker

A lightweight **web-based carbon emission calculator** built using Python and SQLite.  
This application helps users estimate **CO₂ emissions from travel activities** and provides **eco-friendly suggestions** to reduce environmental impact.

---

## 🚀 Features

- Calculates carbon emissions based on travel data
- Inputs required:
  - Distance travelled (km)
  - Mode of transport (Car / Bike / Bus)
  - Fuel type (Petrol / Diesel / Electric)
  - Engine capacity (CC)
- Stores results in **SQLite database**
- Displays **emission history with date and time**
- Provides **environmental suggestions** based on emission level
- Simple **single-source Python implementation**

---

## 🛠 Technologies Used

- SQLite
- HTML
- CSS
- HTTPServer
